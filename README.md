# Research-Assignment
This assignment was mainly about understanding the fundamentals of data, artificial intelligence, and how they are used in the real world. It covered key concepts like Natural Language Processing, cloud computing, REST APIs, and machine learning models such as large language models. Through these topics, I learned how technology can process human language, store and manage data online, and allow different systems to communicate with each other. The assignment also introduced practical programming ideas like the map function, helping me see how data can be transformed and handled efficiently in code.
Another important part of the assignment focused on the ethical side of data and AI. It explored issues like bias in AI systems and how data can be misused if not handled responsibly. This made me realize that working with data is not just about technical skills, but also about making fair and ethical decisions. Overall, the assignment helped me connect theory with real-life applications, showing how data and AI already influence everyday tools and how they will continue to shape future careers and society.
